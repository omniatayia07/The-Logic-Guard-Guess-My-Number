# 🎯 Guess My Number Game (لعبة تخمين الرقم)

أهلاً بكم في مشروعي البسيط! أنا عملت اللعبة دي بلغة **بايثون (Python)**، وهي عبارة عن لعبة تفاعلية بتخلي الكمبيوتر يختار رقم في السّر، واللاعب بيحاول يخمن الرقم ده في عدد محاولات محدد.

---

## 🧐 فكرة اللعبة والخطوات البرمجية

أنا صممت الكود ده ورتبت خطواته كالتالي:

1. **اختيار الرقم السري:** أنا استوردت مكتبة `random` وخليت الكمبيوتر يختار رقم عشوائي من **1 إلى 10** عشان اللاعب يحزره.
2. **تحديد الفرص:** أنا حددت عدد المحاولات بـ **5 فرص بس**، عشان اللعبة يكون فيها تحدي ومينفعش اللاعب يفضل يخمن للأبد.
3. **حلقة التكرار (While Loop):** أنا عملت لوب عشان اللعبة تفضل شغالة وتطلب من اللاعب يدخل تخمينه طول ما لسه معاه فرص متبقية.
4. **تأمين الكود وتفادي الأخطاء:** أنا استخدمت نظام الـ `try-except` عشان لو اللاعب استهبل وكتب حروف أو رموز بالغلَط بدل الأرقام، البرنامج ما يبوظش ويقفل؛ لا، بيطلع له رسالة تنبيه وخليه يكمل عادي من غير ما يخسر فرصة.
5. **خصم المحاولات:** أنا خليت العداد يخصم محاولة فوراً بمجرد ما اللاعب يكتب رقم صح ويدوس Enter.
6. **مقارنة الأرقام والتوجيه:** 
   - لو رقم اللاعب طلع صح، الكود بيقوله إنه كسب وبيوقف اللعبة فوراً باستخدام أمر `break`.
   - لو التخمين غلط، الكود بيساعد اللاعب ويغششه: لو رقمه أصغر بيقوله جرب رقم أعلى (`Try higher`)، ولو أكبر بيقوله جرب رقم أقل (`Try lower`).
7. **نهاية الجيم (Game Over):** أنا عملت شرط أخير برا اللوب خالص عشان لو الفرص خلصت واللاعب معرفش يوصل للحل، يطبع له "جيم أوفر" ويكشف له الرقم السري اللي كان مخفي.

---

## 🛠️ التكنولوجيا المستخدمة

* **اللغة:** Python 3
* **المكتبات:** `random` (مكتبة مدمجة لتوليد الأرقام العشوائية).

---

## 🚀 طريقة تشغيل اللعبة

عشان تشغل اللعبة عندك، اتبع الخطوات دي:

1. اتأكد إن عندك بايثون متطبب على الجهاز.
2. خد الكود كوبي واحفظه في ملف سميه مثلاً `game.py`.
3. افتح التيرمنال (Terminal) أو الـ CMD واكتب الأمر ده عشان تشغل اللعبة:

```bash
python game.py
```
